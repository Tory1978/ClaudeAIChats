# ClaudeAIChats
Шаг 1 — Экспортировать чаты из Claude.ai
Открой claude.ai
Нажми на своё имя / аватар в левом нижнем углу
Выбери Settings (Настройки)
Найди раздел "Privacy"
Нажми кнопку "Export data"
На почту придёт письмо со ссылкой — скачай архив .zip
Внутри архива будет папка с файлами .json — это твои чаты.

Шаг 2 - Загрузить на GitHub
Открыть закладку Code 
Убедись, что ты находишься в ветке archive (кнопка должна показывать archive)
Если нет — нажми на кнопку с названием ветки и выбери archive из списка
Нажми "Add file" → "Upload files"
Нажми "Commit changes"

Шаг 5 — Удалить файлы из ветки (опционально)
Переключись на ветку 
Найди загруженные файлы чатов
Открой файл → нажми иконку корзины (🗑) справа вверху
Нажми "Commit changes"
